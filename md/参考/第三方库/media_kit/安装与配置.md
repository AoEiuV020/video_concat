# 安装与配置

## 包结构

[media_kit](https://github.com/media-kit/media-kit) 拆分为多个包，按需组合：

| 包 | 版本 | 用途 |
|----|------|------|
| `media_kit` | ^1.2.6 | 核心：Player, Media, Playlist |
| `media_kit_video` | ^2.0.1 | 视频渲染：VideoController, Video widget |
| `media_kit_libs_video` | ^1.0.7 | 预编译原生依赖（libmpv + ffmpeg），含视频+音频 |
| `media_kit_libs_audio` | ^1.0.7 | 仅音频的原生依赖（更小体积） |

视频播放的标准依赖：

```yaml
dependencies:
  media_kit: ^1.2.6
  media_kit_video: ^2.0.1
  media_kit_libs_video: ^1.0.7
```

> `media_kit_libs_video` 和 `media_kit_libs_audio` 不要同时使用。

## 平台支持

| 平台 | 视频 | 音频 | 最低要求 |
|------|------|------|----------|
| macOS | ✅ | ✅ | macOS 10.9 |
| Windows | ✅ | ✅ | Windows 7 |
| Linux | ✅ | ✅ | 现代发行版 |
| Android | ✅ | ✅ | Android 5.0 |
| iOS | ✅ | ✅ | iOS 9 |
| Web | ✅ | ✅ | 现代浏览器 |

## SDK 要求

- Dart SDK >= 3.1.0
- Flutter SDK >= 3.0.0（原生依赖包）

## 初始化

在 `main()` 中必须调用：

```dart
void main() {
  WidgetsFlutterBinding.ensureInitialized();
  MediaKit.ensureInitialized();
  runApp(const MyApp());
}
```

## 原生依赖分发

每个平台有独立的原生依赖包（[`libs/` 目录](https://github.com/media-kit/media-kit/tree/main/libs)）：

| 平台 | 包名 | 包含内容 |
|------|------|---------|
| macOS | `media_kit_libs_macos_video` | 预编译 libmpv + ffmpeg dylib |
| Windows | `media_kit_libs_windows_video` | 预编译 libmpv + ffmpeg dll |
| Linux | `media_kit_libs_linux` | Linux 平台依赖入口；构建期仍需系统提供 `mpv` / `libmpv-dev` |
| Android | `media_kit_libs_android_video` | 预编译 libmpv + ffmpeg so |
| iOS | `media_kit_libs_ios_video` | 预编译 libmpv + ffmpeg xcframework |

统一入口 `media_kit_libs_video` 自动依赖对应平台包。

## macOS 特殊配置

无额外配置。`media_kit_libs_macos_video` 通过 CocoaPods 自动集成。

## Linux 特殊配置

Linux 构建阶段会通过 `pkg-config` 查找 `mpv`，因此即使已经依赖 `media_kit_libs_video`，仍需要先安装系统包：

```bash
sudo apt-get update
sudo apt-get install mpv libmpv-dev
```

如果缺少这些包，CMake 常见报错是：

- `Target "media_kit_video_plugin" links to: PkgConfig::mpv but the target was not found`
- `Target "media_kit_native_event_loop" links to: PkgConfig::mpv but the target was not found`

## Android 特殊配置

1. 在 `android/app/build.gradle` 中添加：

```groovy
android {
    packagingOptions {
        pickFirst 'lib/**/libc++_shared.so'
    }
}
```

2. 建议启用 `--split-per-abi` 或使用 app bundle 减小包体积。

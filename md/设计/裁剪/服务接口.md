# 服务接口

裁剪功能需要以下服务能力，均基于 FFmpeg/FFprobe 命令行工具。

## 关键帧探测

### 局部探测（推荐）

在指定时间窗口内查找关键帧，不遍历全文件。

| 参数 | 说明 |
|------|------|
| filePath | 视频文件路径 |
| startTime | 查询窗口起点（微秒） |
| endTime | 查询窗口终点（微秒） |
| **返回** | 关键帧时间戳列表（微秒），按升序排列 |

底层命令（时间参数需格式化为秒字符串）：

```bash
ffprobe -read_intervals {startTime}%{endTime} \
  -select_streams v:0 -skip_frame nokey \
  -show_frames -show_entries frame=pkt_pts_time \
  -of csv=p=0 input.mp4
```

### 全量探测

获取视频中所有关键帧时间戳。适用于需要展示完整时间线的场景。

| 参数 | 说明 |
|------|------|
| filePath | 视频文件路径 |
| **返回** | 关键帧时间戳列表（微秒），按升序排列 |

底层命令：

```bash
ffprobe -select_streams v:0 -skip_frame nokey \
  -show_frames -show_entries frame=pkt_pts_time \
  -of csv=p=0 input.mp4
```

## 帧预览提取

提取指定时间点的视频帧画面，用于裁剪预览。

| 参数 | 说明 |
|------|------|
| filePath | 视频文件路径 |
| timestamp | 目标时间（微秒），应为关键帧时间 |
| maxWidth | 预览图最大宽度（像素），高度按比例缩放 |
| **返回** | JPEG 图像字节数据 |

底层命令（timestamp 需格式化为秒字符串，管道输出不产生临时文件）：

```bash
ffmpeg -ss {timestamp} -i input.mp4 \
  -vframes 1 -q:v 5 -vf scale={maxWidth}:-1 \
  -f image2pipe -vcodec mjpeg pipe:1
```

由于只提取关键帧画面（`-ss` 在 `-i` 之前直接跳转），耗时约 50~80ms。

## 文件列表生成（扩展）

现有的文件列表生成逻辑需要扩展，支持为每个条目追加 inpoint/outpoint 指令。

### 输入

视频项列表，每项包含：

| 字段 | 说明 |
|------|------|
| filePath | 文件路径 |
| trimConfig | 裁剪配置（可选），包含 0~N 个片段 |

### 输出规则

| 场景 | 生成内容 |
|------|---------|
| 无 trimConfig 或空片段 | 单条 `file 'path'` |
| 有片段 | 每个片段一条 `file 'path'` + `inpoint` + `outpoint` |
| inpoint = 0 | 省略 `inpoint` 行 |
| outpoint = 视频末尾 | 省略 `outpoint` 行 |

### 伪代码

```
function buildFileList(items):
  for item in items:
    if item.trimConfig 为空或无片段:
      输出 "file '{item.filePath}'"
    else:
      for segment in item.trimConfig.segments:
        输出 "file '{item.filePath}'"
        if segment.inpoint > 0:
          输出 "inpoint {segment.inpoint}"
        if segment.outpoint < item.duration:
          输出 "outpoint {segment.outpoint}"
```

## 视频时长获取

裁剪页面需要视频总时长来设置进度条范围。

| 参数 | 说明 |
|------|------|
| filePath | 视频文件路径 |
| **返回** | 时长（微秒） |

可复用现有探测接口的 format.duration 字段，或单独查询：

```bash
ffprobe -select_streams v:0 \
  -show_entries format=duration \
  -of csv=p=0 input.mp4
```

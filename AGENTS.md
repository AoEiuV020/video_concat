# AGENTS.md

## Commands

```bash
# 依赖
melos bs                         # bootstrap，获取所有包的依赖

# 代码生成（freezed / riverpod_generator）
melos gen

# 静态分析
melos analyze

# 测试 → 加载 flutter-test skill

# 提交前格式化
melos precommit                  # = melos fix + melos format + melos sort

# 构建 → 加载 flutter-build skill
```
